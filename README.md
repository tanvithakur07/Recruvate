# Recruvate Website

## Description
Recruvate is a multi-page responsive corporate website developed using HTML, CSS, and JavaScript. The platform represents an HR & career development company offering recruitment services, skill development programs, and professional courses.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
-Multi-page structured website
-Responsive design (mobile-friendly)
-Interactive navigation bar
-Course listing and individual course pages
-Login & Registration forms
-Testimonials and Contact section

## How to Run the Project
1.Download or clone the repository
2.Open the project folder
3.Open index.html in your browser
